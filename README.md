# CNN-Brain-Tumor-Classification-Project
Brain tumor classification using a basic CNN model. 

We have used a dataset from kaggle, 
https://www.kaggle.com/datasets/masoudnickparvar/brain-tumor-mri-dataset/data
to create a simple brain tumor classification AI tool. 

We named it NeuroScan. 
All the codes that we have used for the AI are linked below. 

https://colab.research.google.com/drive/1HX40GH2luB6ltUhuitH8XbF5yeEEif1w?usp=sharing

Our AI model was simple, yet it successfully reached .993 accuracy.
It is a demonstration of how easily AI tools can be used to help medical decisions.
